My latest project

TalkingTools

TalkingTools is a learning platform that was developed to help foreign students of the Kunstakademie Stuttgart to work with tools at their workshops. It's a responsive single page webapp (main focus on mobile version) that provides video tutorials and safety instructions in form of text and pictures about workshops and tools. The project was created in cooperation with employees of the ABK Stuttgart, but was fully implemented by myself. It was written with the JavaScript framework Svelte and is hosted on an Apache webserver.
